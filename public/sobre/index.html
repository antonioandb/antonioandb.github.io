Olá! 👋#

Meu nome é Antonio Barros.

Gosto de aprender como as coisas funcionam e utilizo este espaço para documentar tudo o que aprendo durante minha jornada no mundo do Linux, desenvolvimento de software, infraestrutura e software livre.

Em vez de deixar minhas anotações espalhadas em arquivos pessoais, decidi transformá-las em artigos públicos para que outras pessoas também possam aprender com elas.

Meu objetivo é construir uma base de conhecimento organizada, prática e útil, tanto para mim quanto para quem estiver estudando os mesmos assuntos.
